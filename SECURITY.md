## Reporting a Vulnerability

We take the security of this project seriously. If you discover a vulnerability, please report it to us immediately.

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please report them via email to:
**security@strategicautomation.co.uk**

Please include:
- A description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact.

We will acknowledge your report within 48 hours and provide an estimated timeframe for resolution.
